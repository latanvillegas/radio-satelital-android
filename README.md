# radio-satelital-android
apk al 100% nativa en klotin de Radio Satelital

## Conectar Firebase

1. En Firebase Console, crea (o abre) tu proyecto Firebase.
2. Registra una app Android con el package name: com.app.radiosatelital.
3. Descarga el archivo google-services.json.
4. Copia ese archivo en app/google-services.json.
5. En Firebase Console, habilita Authentication (Anonymous y/o Email/Password) y Firestore Database.
6. Ejecuta la app desde Android Studio o con Gradle.

Nota:
- El archivo google-services.json se mantiene fuera de Git por seguridad.
